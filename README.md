# Rei-Bot

A Discord Bot that is used to play music

Written in Python, not configured for public usage.  

# Requirements

Application is developed on ARM Mac, and is currently not tested on other platforms.

- macOS Ventura 13.4
- Python 3.11.4
- discord.py 2.3.1
- youtube_dl 2021.12.17
- ffmpeg 6.0
